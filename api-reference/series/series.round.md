# Series.round

Round each value in a Series to the given number of decimals.



 **parameter:** {dp} Number, Numbers of Decimal places to round to

            **return:** {Series}

**Example**

```javascript
let data1 = [30.21091, 40.190901, 3.564, 5.0212]
let sf = new Series(data1)
sf.round(1)
```
